What is the DOM?

The Document Object Model (DOM) is a programming interface that allows developers to interact with and manipulate HTML and XML documents.

Why is the document called the Object Model?

Document is modeled of object and document in not only the structure of an object but it also includes the behavior of document structure and object that contains the elements like HTML tags and their attributes.

What are the HTML DOM methods involved?

The following are HTML DOM methods that are mostly used.

  • getElementById(idName)
  • getElementsByClassName(className)
  • getElementsByTagName(tagName)
  • appendChild(element)
  • removeChild(element)
  • document.appendChild(element)
  • createChild(element)
  • replaceChild(new, old)
